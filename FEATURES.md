# Corridor Features Overview

Corridor offers a comprehensive suite of features across three tiers, designed to scale from individual users to enterprise organizations.

## 🎯 Tier System

### 🆓 Free Tier
**Perfect for individuals and small teams**
- Up to 5 users
- Basic payment processing
- Standard support
- Core features with usage limits

### 💼 Pro Tier ($29/month)
**Ideal for growing businesses**
- Up to 50 users
- Advanced payment rails
- Priority support
- Enhanced features and integrations

### 🏢 Premium Tier ($99/month)
**Enterprise-grade capabilities**
- Unlimited users
- White-label options
- Dedicated support
- Full feature access and customization

## 🤖 MCP Integration (AI Workflows)

### Available Tools (13 Total)

**Business Operations:**
1. `create_invoice` - Generate and send invoices
2. `process_payment` - Handle payment transactions
3. `check_balance` - Query account balances
4. `generate_report` - Create financial reports

**Social Features:**
5. `create_goal` - Set up crowdfunding goals
6. `contribute_to_goal` - Make goal contributions
7. `share_expense` - Split payments among users
8. `send_payment` - P2P money transfers

**EWA Management:**
9. `request_advance` - Employee wage advances
10. `approve_advance` - Manager approval workflows
11. `sync_payroll` - ERP system synchronization

**Analytics & Insights:**
12. `analyze_spending` - Financial pattern analysis
13. `predict_cashflow` - AI-powered forecasting

### Tier Access:
- **Free**: 3 tools (basic payment, balance, invoice)
- **Pro**: 8 tools (adds social features, basic EWA)
- **Premium**: All 13 tools + custom tool development
